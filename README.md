# MyContacts

Projeto criando com: spring boot, flyway para criar as migrations e spring data jpa.

- Para rodar o projeto, basta criar um banco de dados postgres, com docker ou com pgadmin, e setar as variáveis de ambiente no application.properties.
- Nesse projeto foram abordados conceitos como estruturação em camadas, exceções personalizadas e com retorno padronizado, validações de entrada, e queries personalizadas com @Query.
- Próximos passos: refatorar alguns códigos repetitivos, utilizar docker para subir a api em uma vps.
